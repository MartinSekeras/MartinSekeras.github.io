of boxes that can be fit onto variety of pallets. In no way I take responsibility for any

Updates

03 June 2023

Update version 0.3a, added a lot of functionality to pallet stacker.
For one it is now possible to calculate additional orientations. This means that
the calculator can detect and orient boxes on pallet to fit more. Such as 3 boxes
laying flat and 1 box standing up on it's end at the endge of a pallet.
This is a beta feature and only reflect double pallets and may not work 100%

 Update version 0.3b, New quick update on features i wanted to add before.
The web app can now handle error correcting when nothing is inputted into fields, it
can also detect overweight pallets and alert user if the pallet may be too heavy
for racking, depending on lenght of pallet and weight the user inputted. These
variables are hard coded will vary as every deployment is different, optional input field planned.

04 June 2023

Update version 0.4, New day new update, fairly small update today. The web app is
now simplified, no longer pallet spec is user inputtable but simple checkboxes with
predetermined values. Future update will feature custom checkbox with user
selectable values but for now for simplicity sake it was made with 2 options.
